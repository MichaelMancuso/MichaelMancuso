    How to read this chart

    Operational Risk blends exploit reality, blast radius, remote reachability, and how likely the issue is to matter to a defender right now.

    Exploit Momentum emphasizes current exploitation activity and adversary interest. Internet Exposure favors browser and edge-facing software. Patch Urgency prioritizes how quickly defenders should move based on exploitation, exposure, and likely impact.

    Methodology note: this dashboard is intentionally curated and not exhaustive. It focuses on a small set of recent, high-signal cases so the chart stays readable and useful in a portfolio or briefing context.
